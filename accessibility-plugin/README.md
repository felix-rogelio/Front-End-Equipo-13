# Plugin de Accesibilidad

Esta carpeta agrupa dos versiones del plugin de accesibilidad:

- `react-version/` — versión para aplicaciones React.
- `portable-version/` — versión independiente de React para cualquier web.

## Estructura

- `react-version/`
  - `AccessibilityToolbar.jsx`
  - `AccessibilityToolbar.css`
  - `README.md`
- `portable-version/`
  - `accessibility-plugin.js`
  - `README.md`

## Instrucciones

Cada versión tiene su propia carpeta con el código necesario y un `README.md` con pasos de instalación específicos.

- Usa `react-version/README.md` para instalar la versión React.
- Usa `portable-version/README.md` para instalar la versión portable.

## Qué ofrece el plugin

- Aumentar texto
- Reducir texto
- Alto contraste
- Fuente legible
- Reducir movimiento
- Resaltar foco
- Lectura en voz alta
- Restablecer
